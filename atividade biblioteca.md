Para a biblioteca ser utilizada corretamente deve-se usar um alyance, ou seja, um apelido. Cada bibliotecas possui uma função sendo elas 

**BIBLIOTECA MATEMATICA --> m**

Essa biblioteca apresenta uma serie de funções matemáticas mais para usar no dia a dia. Nela contem a função como PI, Arredondar, cosseno, logaritmo, maior numero, menor numero, potencia, raiz, seno, tangente e valor absoluto. 

**BIBLIOTECA** **TEXTO --> tx**

Essa biblioteca apresenta funções como caixa alta ou baixa, substituir por um texto alternativo, obter caracteres ou contar a quantidade de delas, extrair um subtexto e posicionar um texto dentro de uma cadeia. Ela contem funções para a manipulação de algum texto

**BIBLIOTECA UTIL --> u**

Essa biblioteca pode pedir para aguardar comandos em milissegundos, sortear algum número, números de linhas ou de elementos ou de colunas e o tempo decorrido. Ela contém diversas funções utilitárias
